Word Guess Game

A fun and challenging word guessing game where players try to identify a hidden word by guessing letters. Built with vanilla JavaScript and featuring a bold neo-brutalist design aesthetic.

Features

300+ Word Dictionary - Words ranging from 7-10 letters for consistent difficulty
Dual Input Methods - Click on-screen keys OR type with your keyboard
Visual Feedback - Shake animations for wrong guesses, pop effects for correct ones
Progress Tracking - Clear display of remaining guesses and used letters
Bold Neo-Brutalist UI - Unique design with heavy borders, offset shadows, and vibrant colors
Fully Responsive - Works perfectly on desktop, tablet, and mobile devices

How to Play

Start the Game - A random word between 7-10 letters is selected
Make Your Guess - Click a letter on the keyboard or type on your physical keyboard
Track Progress - Watch the word reveal as you guess correct letters
Manage Attempts - You have 6 wrong guesses before game over
Win or Lose - Guess the complete word before running out of attempts!
